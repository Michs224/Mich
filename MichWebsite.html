Portfolio

Throughout my professional journey, a diverse range of challenging and innovative projects have materialized, reflecting my unwavering dedication to developing effective technological solutions that deliver significant positive impact.

Personal Chatbot LLM_RAG

Personal chatbot project using Large Language Models (LLM) and Retrieval Augmented Generation (RAG). The chatbot provides detailed answers by leveraging knowledge from PDF documents, employing LlamaCPP for LLM integration and llama_index for efficient document handling.

Patient's Condition Classification Using Drug Reviews

Patient condition classification using drug reviews applies multiple machine learning models such as Passive Aggressive Classifier, Random Forest, XGBoost, and etc. The project also applies text processing techniques including CountVectorizer, TFIDFVectorizer, and Word2Vec to analyze user reviews from a dataset.

Credit Card Fraud Detection Model

Machine learning model to detect credit card fraud using various algorithms such as Logistic Regression, Decision Tree, Random Forest, and XGBoost. This dataset is highly imbalanced, so several resampling techniques are applied to balance the data

Credit Risk Model

Credit risk prediction model aimed at predicting customers who are at potential risk of defaulting on their loans based on historical loan data.

Movie Recommendation System

This ML project utilizes a Content-Based Recommender System for movies, using features like titles, genres, and posters to provide recommendations based on the chosen film.

Image Classification of Billionaires

This project contains a classification of billionaires, namely 20 billionaires.

LibraLink: Smart Attendance System with Book Recommendation

LibraLink: An advanced library system integrating facial recognition for attendance, using CNN or LBPH algorithms based on facial data patterns. It also features a book recommendation service, matching user preferences through the K-Nearest Neighbors (KNN) algorithm.

Speech Emotion Recognition

Speech Emotion Recognition (SER) using a Convolutional Neural Network (CNN). The project utilized datasets such as CREMA, RAVDESS, SAVEE, and TESS to train the model to classify various emotions from audio recordings.

Simple Personal Chatbot

This simple personal chatbot project uses Artificial Neural Networks (ANNs) to process user queries, utilizing Flask and TensorFlow for development. It performs data preprocessing by tokenizing, removing punctuation and stopwords.

Food and Beverages_OOPs Concepts

OOPs concept using Java and connect to MySQL Database in Food and Beverage (FnB).
